# E-Commerce
https://allpha23.github.io/e-commerce/

# Sobre o projeto
Este é um projeto de um E-Commerce totalmente responsivo, desenvolvido utilizando a API do Mercado Livre. O objetivo deste projeto é permitir que os usuários possam fazer compras diretamente através do seu app ou site, consumindo os dados da API do Mercado Livre.

# Funcionalidades
- Pesquisar produtos disponíveis no Mercado Livre;
- Detalhar um produto selecionado;
- Adicionar produtos ao carrinho de compras;

# Tecnologias utilizadas
- ReactJS
- JavaScript
- Typescript
- Tailwind
- Axios
- Redux
- React-router-dom
- Eslint

# Requisitos
Antes de iniciar o projeto, é necessário possuir as seguintes dependências:
- Node.js
- npm ou yarn

# Instalação
```bash
# clonar repositório
git clone git@github.com:allpha23/e-commerce.git

# entrar na pasta do projeto
cd e-commerce

# instalar dependências
npm install
```
# Uso
Para iniciar o projeto, execute o seguinte comando:
```bash
# executar o projeto
npm run dev
```
O projeto será executado na porta 5173.
<br>
Acesse http://localhost:5173 para utilizar o E-Commerce.
